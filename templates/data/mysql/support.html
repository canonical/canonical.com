Multiple packaging formats

We provide maintained and supported MySQL packages in the most popular formats:

Debs

We offer deb packages for a seamless integration with your Ubuntu installations.
Our MySQL debs help you:

  • Deploy a working MySQL instance in a single command
    (sudo apt install mysql-server)
  • Easily add some of the most used packages and tools such as xtrabackup and MySQLTuner

Snaps

Get snaps for added confinement and atomic updates on all major Linux distributions. We support two options:

  • MySQL-chiselled snap for the smallest possible footprint and attack surface
  • MySQL snap for a complete solution integrating the most popular MySQL plugins, backup and monitoring tools

Containers

We deliver OCI-compliant images for your containerized environment. We provide several supported options:

  • MySQL-chiselled rock for the smallest possible footprint and attack surface
  • MySQL rock for a complete solution integrating the most popular MySQL plugins, backup and monitoring tools
