Multiple packaging formats

We provide maintained and supported PostgreSQL packages in the most popular formats:

Debs

We offer deb packages for a seamless integration with your Ubuntu installations.
Our PostgreSQL debs help you:

  • Deploy a working PostgreSQL instance in a single command
    (sudo apt install postgresql).
  • Easily add extensions to your PostgreSQL installations.
  • Easily manage multiple PostgreSQL instances in the same node using pg_* utilities such as pg_lsclusters .

Snaps

Get snaps for added confinement and atomic updates on all major Linux distributions . We support two options:

  • PostgreSQL-only snap for the smallest possible footprint and attack surface.
  • Charmed PostgreSQL snap for a complete solution integrating the most popular PostgreSQL extensions, backup, monitoring and high availability tools.

Containers

We deliver OCI-compliant images for your containerized environment. We provide several supported options:

  • PostgreSQL-only regular container.
  • PostgreSQL-only chiselled rock for the smallest possible footprint and attack surface.
  • Charmed PostgreSQL rock for a complete solution integrating the most popular PostgreSQL extensions, backup, monitoring and high availability tools.
