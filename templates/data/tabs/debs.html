Debian Package Archives

Where applicable, we offer deb packages for a seamless integration with your Ubuntu installations.

Debian (deb) packages are archive files that contain all the necessary components for a specific piece of software. They generally include binaries, executables and libraries, configuration files and install and removal scripts.
