Integrating all the cloud-native management functions into one platform enables repeatable and reliable cluster deployments and operations.

Starting with the bare metal provisioning, MAAS provides comprehensive automation to effortlessly commission, deploy, manage and retire physical servers. It relies on industry standards infrastructure-as-code automation frameworks such as Terraform and Kubernetes Cluster API.

Terraform enables consistency and reproducibility across the hybrid cloud stack and lets you integrate with CI/CD pipelines.

Kubernetes Cluster API (CAPI) automates declarative deployment of clusters and manages full-stack environments with both VMs and containers.
