Cloudify your data centre

Canonical enables you to run your data centre like a cloud. We provide a unified, cloud-native infrastructure capable of seamlessly integrating virtualised network functions (VNFs) and cloud-native network functions (CNFs).

This integration maximises the strengths of each platform, ensuring that telecom companies can meet the most stringent 5G Core requirements with optimal performance and flexibility.

Virtualise key network functions with OpenStack

OpenStack is an open source cloud platform which plays a pivotal role in modern telecom networks. It provides the infrastructure necessary to host virtual network functions (VNFs) efficiently, enabling telecom companies to meet 5G Core requirements with flexibility and high performance.

What's included

  • Enhanced Platform Awareness (EPA)
  • Service Chaining and Integration
  • Life Cycle Management
  • Scalability and Flexibility

Read more about Canonical OpenStack ›

Orchestrate your Core with Kubernetes

As a proud member of the Cloud Native Computing Foundation (CNCF), Canonical uses Kubernetes to manage cloud-native network functions (CNFs) in telecom networks. Kubernetes is the leading container orchestration platform and automates deployment, scaling, and operations.

What's included

  • Automated scaling
  • Self-healing
  • Rolling updates
  • Enhanced Platform Awareness (EPA)
