Performant user plane

Continually optimize performance with enhanced platform awareness (EPA) features targeted at data plane performance improvement, such as DPDK, Hugepages and CPU pinning. Charmed Aether SD-Coreâ€™s user plane function (UPF) runs on Kubernetes as a cloud-native CNF or on bare metal for best performance.

Network slicing

Deploy and run multiple user plane instances as network slices and associate device groups to those slices. This allows for more efficient use and management of dedicated user plane and platform resources allocated to different sets of devices.
