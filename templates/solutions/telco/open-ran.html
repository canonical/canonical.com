Ubuntu for Open RAN

To achieve carrier-grade Open RAN deployments, virtual RAN software requires an operating system that is robust, immutable and battle-tested in the Telco industry. Choose Ubuntu as your OS and benefit from the low latency, security and enhanced platform features that telcos need to deliver next-generation connectivity.

Guaranteed bounded low-latency

Real-time Ubuntu ensures latency in data processing is bounded and low. Ubuntu's Real-time kernel prioritizes Open RAN networking software workloads and provides streamlined packet processing. This brings support for even the most stringent use cases, categorized as ultra-reliable low latency communication (URLLC) applications.

Immutable

Telco far-edge clouds require tamper-proof deployments that are self-healing, with minimal footprint. Ubuntu Core, the embedded Linux operating system for devices, is secure by design, and is immutable thanks to its containerised architecture. Implement your Open RAN system and run your virtual RAN software with the confidence that only your system admins can modify your remote systems. If you are a radio unit vendor, check out our embedded Linux page for more resoures.

Optimized

Thanks to our partnership with an extensive range of hardware vendors, Ubuntu is optimized to run on all popular hardware systems, including bare metal servers, the latest cutting-edge silicon, and enhanced platform components like DPUs and IPUs. This means that Ubuntu delivers the best possible performance on all mainstream hardware, ready for your Open RAN deployments.

Secure and compliant

Telecom networks are considered to be part of critical national infrastructure. Open RAN deployments must be both safeguarded and compliant with all telecom security standards. With Ubuntu Pro, operators get the most comprehensive security subscription service for open source software. Open RAN fleet of servers get live CVE patches and streamlined updates and upgrades without downtime. Ubuntu Pro ensures compliance with all telecom security standards, and hardening for the minimum attack surface.

Transparent

We make sure that you have the right tools to manage your fleet of RAN systems at scale. With Canonical Landscape, you get visibility on all your Ubuntu fleet, and can perform security patches, auditing, and access management at scale across your multiple RAN sites.

Enhanced platform features

Whichever workload you run at your edge-clouds, be it virtual RAN functions for Open RAN or edge computing software, Ubuntu detects and supports the most advanced hardware. Our vendor partnership programme makes it possible to support a variety of hardware, such as function accelerators, GPUs and network cards.

Vendor neutral products

We work with all major hardware vendors to optimize our infrastructure products, ensuring that they are ready to run Open RAN CU, DU and RU at the highest performance and reliability levels.
