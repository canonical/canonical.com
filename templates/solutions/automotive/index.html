A modern cloud-based environment for demanding Android development teams: enable faster software creation and testing at higher scale, while saving on OPEX.

Anbox Cloud allows teams to develop, debug and test a virtual infotainment device running in the cloud, rather than on the developers' desk.

It supports Android tools (like Android Studio and ADB) and is available on major public clouds as well as private clouds. Anbox Cloud runs on both x86 and Arm based servers as well as Intel, AMD and NVIDIA GPUs.

But more importantly, Anbox Cloud is designed to support the highest number of Android developers per server, providing a very low cost per user.
