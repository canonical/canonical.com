True open source composability

Open source enables you to develop and customize applications to meet your unique business requirements. Consuming available open source lets you move faster. But maintaining open source apps can be costly and time-intensive.

Canonical enables you to benefit from open source in an affordable and convenient way, without lock-in or hidden costs. We support the most popular cloud-native open source projects, offering you upstream innovation with the security assurances you need.

You can use standard APIs and choose among different packaging formats to suit your needs.

Developers love our tools

Access the widest open source library with a consistent security maintenance commitment, and build on FIPS compliant images. Benefit from the same consistent experience everywhere — it's still the same Canonical software you'd find in your laptop, bare metal machines and VMs.

  • Build hardened and optimized containers with Rockcraft – so solid, we call them rocks.
  • Chisel your containers – trim out what you don't need for maximum efficiency. Chisel is a tool to create minimal Ubuntu containers comprising only the application and its runtime dependencies. It's the Canonical take on distro-less containers.
  • Automate application lifecycle management with charms – let Kubernetes operators do the heavy lifting.
